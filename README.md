# Sistema de Gestão de Vendas e Estoque (Java)

Projeto desenvolvido para prática de lógica de programação e construção de um sistema simples utilizando Java.

## Sobre o projeto

Este sistema simula a operação de um pequeno comércio, permitindo registrar vendas, controlar o estoque de produtos e visualizar relatórios diretamente via terminal.

## Funcionalidades

- Registro de vendas por cliente
- Controle de estoque em tempo real
- Reposição de produtos
- Histórico de vendas por item
- Identificação da maior venda realizada
- Exibição do estoque completo
- Interface interativa via terminal

## Conceitos aplicados

- Estruturas de controle (if, switch, loops)
- Organização de código em funções
- Manipulação de variáveis globais
- Entrada de dados com Scanner
- Validação de regras de negócio (estoque e vendas)

## Tecnologias utilizadas

- Java
- Execução em ambiente de console

## Estrutura do sistema

O sistema trabalha com:

- Cadastro de produtos (nome, preço e estoque)
- Processamento de vendas com validação de disponibilidade
- Atualização automática de estoque
- Registro de histórico e análise de vendas
